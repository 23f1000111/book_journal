# Book Journal

A personal reading log application to track your reading journey, manage your wishlist, and visualize your reading habits.

![Book Journal Preview](https://via.placeholder.com/800x400?text=Book+Journal+Preview)

## Features

- **Personal Journal**: Log every book you read with detailed reviews, ratings, dates, and favorite quotes.
- **Wishlist**: Keep track of books you "Want to Read" with cover images and purchase links.
- **Analytics**: Visualize your reading habits with charts for books per month, genre distribution, and rating spread.
- **Export Data**:
  - **CSV Export**: Download your raw data for spreadsheets.
  - **PDF Export**: Generate a beautiful, printable PDF of your reading journal with book covers.
- **Secure**: Powered by Firebase Authentication and Firestore for secure cloud storage.

## Technologies

- **Frontend**: Vanilla HTML, CSS, JavaScript
- **Backend**: Firebase (Auth, Firestore)
- **Libraries**:
  - [Chart.js](https://www.chartjs.org/) for analytics visuals.
  - [html2pdf.js](https://github.com/eKoopmans/html2pdf.js) for PDF generation.
  - [FontAwesome](https://fontawesome.com/) for icons.

## Setup

1.  Clone the repository.
2.  Open `index.html` in your browser (or serve with a local server like Live Server).
3.  **Note**: You will need to provide your own `firebase-config.js` with your API keys to connect to your own Firebase project.

## deployment

This project is ready for deployment on **GitHub Pages**, **Netlify**, or **Vercel**.
